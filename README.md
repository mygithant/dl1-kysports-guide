# dl1-kysports-guide

这是一个主要用于归档和发布多个独立 HTML 页面的仓库。这些页面以离线或半离线方式提供信息参考，不针对任何特定域名或网站。

## 项目简介

本仓库收纳了一系列独立的 HTML 文档，内容涵盖体育相关的参考资料、指南、数据汇总等。每个页面均以静态 HTML 形式存放，便于直接浏览、本地保存或部署到静态托管服务。

所有内容均以信息整理和归档为目的，不包含动态后端或用户交互功能。

## 目录结构

```
dl1-kysports-guide/
├── index.html          # 仓库主页/导航页
├── pages/              # 存放各独立 HTML 页面的目录
│   ├── guide-01.html
│   ├── guide-02.html
│   └── ...
└── assets/             # 公共资源（图片、样式表、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库的入口页面，通常包含页面列表或简要导航。
- `pages/`：所有独立 HTML 页面存放于此。每个文件均为完整可用的 HTML 文档。
- `assets/`：存放页面共享的静态资源，如样式文件、脚本、图片等。

## 页面归档说明

- 每个 HTML 页面均为独立归档，可单独打开使用。
- 页面内容可能包含体育领域的统计、规则、历史、术语说明等，不保证时效性或权威性。
- 部分页面可能引用外部资源（如字体、图标库），但核心内容优先以本地或内联方式呈现。
- 归档内容可能来自公开资料整理，仅供学习参考。
